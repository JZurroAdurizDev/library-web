Términos y condiciones

Estos términos regulan el uso de Library Web como plataforma de consulta, gestión y administración de recursos bibliotecarios.

Última actualización: julio de 2026

1. Introducción

Library Web es una aplicación web diseñada para facilitar la gestión de libros, préstamos y usuarios dentro de un entorno bibliotecario.

El acceso y uso de la plataforma implica la aceptación de estas condiciones generales.

2. Uso de la plataforma

La plataforma debe utilizarse únicamente para consultar información bibliotecaria, gestionar préstamos y realizar las operaciones permitidas según el rol asignado a cada persona usuaria.

3. Cuentas de usuario

Cada persona usuaria es responsable de mantener la confidencialidad de sus credenciales de acceso y de utilizar la plataforma de forma adecuada.

Algunas funciones pueden estar limitadas a usuarios autenticados o a perfiles con permisos de administración.

4. Préstamos y gestión de libros

Library Web permite consultar libros disponibles y gestionar préstamos conforme a las reglas definidas por la biblioteca o entidad responsable.

Las fechas de inicio, devolución y cierre de préstamos deberán interpretarse como información operativa dentro de la plataforma.

5. Funciones de administración

Las funciones de creación, edición o eliminación de usuarios, libros y préstamos estarán reservadas a perfiles autorizados.

El uso de estas funciones deberá realizarse con responsabilidad y respetando la integridad de la información almacenada.

6. Uso aceptable

No está permitido utilizar la plataforma para acceder a información no autorizada, alterar datos sin permiso o realizar acciones que puedan afectar al funcionamiento normal del sistema.

7. Limitación de responsabilidad

Library Web se ofrece como herramienta de gestión bibliotecaria. La disponibilidad, exactitud y actualización de los datos dependerá de la configuración y mantenimiento de la plataforma.

8. Cambios en los términos

Estos términos podrán actualizarse para reflejar cambios funcionales, técnicos o normativos relacionados con la plataforma.

9. Contacto

Para cualquier consulta sobre estos términos y condiciones, puedes contactar con el equipo responsable de Library Web.
