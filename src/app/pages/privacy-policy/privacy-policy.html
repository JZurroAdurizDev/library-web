Política de privacidad

En Library Web respetamos la privacidad de las personas usuarias y tratamos la información únicamente para permitir el funcionamiento básico de la plataforma bibliotecaria.

Última actualización: julio de 2026

1. Introducción

Esta política de privacidad describe de forma general cómo se gestionan los datos dentro de Library Web, una aplicación orientada a la consulta de libros, gestión de préstamos y administración básica de recursos bibliotecarios.

2. Datos que se pueden tratar

La plataforma puede gestionar datos básicos de usuario, como nombre, apellidos, DNI, correo electrónico, rol de usuario y datos asociados a la actividad de préstamos.

También pueden tratarse datos relacionados con libros consultados, préstamos activos, préstamos cerrados y fechas asociadas a la gestión bibliotecaria.

3. Finalidad del tratamiento

Los datos se utilizan para permitir el acceso a la plataforma, identificar a las personas usuarias, gestionar préstamos, consultar el catálogo y administrar los recursos disponibles.

En el caso de usuarios administradores, la información también puede utilizarse para gestionar libros, préstamos y cuentas de usuario dentro del sistema.

4. Autenticación y sesión

Library Web utiliza autenticación mediante cookie segura para mantener la sesión de usuario. La aplicación no almacena tokens de autenticación en el almacenamiento local del navegador.

5. Conservación de los datos

Los datos se conservarán mientras sean necesarios para el funcionamiento de la plataforma y para mantener el historial básico de la actividad bibliotecaria.

6. Seguridad

La plataforma aplica medidas razonables para proteger la información gestionada y limitar el acceso a las funciones según el rol de cada persona usuaria.

7. Derechos de las personas usuarias

Las personas usuarias podrán solicitar información sobre los datos asociados a su cuenta, así como pedir su modificación o eliminación cuando corresponda.

8. Contacto

Para cualquier consulta relacionada con esta política de privacidad, puedes contactar con el equipo responsable de Library Web.
